# PetEditor

## 基本概念
### 动作

构成图像播发的基本要素

动作可受外界力影响,速度代表本动作的保持速度

### 事件

以动作为载体,可被触发器触发的一个过程称为事件

事件包含一个动作,初始速度

## 触发器说明
1. 默认触发器,桌宠启动时,使用默认触发器进行动作
2. 碰撞触发器,桌宠碰到环境或其他桌宠时,进行动作
3. 点击触发器,桌宠被点击时触发,进行动作
4. 时长触发器,桌宠到特定时长,进行动作
5. 目的触发器,桌宠移动到目的地或范围内,进行动作

### 触发概率

说明:设置多个事件,当触发时,前个事件未触发,则下个事件触发概率为本事件概率加上前个事件触发概率

举例:碰撞触发器,碰墙,50%的反向,50%的爬墙.则实际碰墙时,未触发反向时,爬墙为100%触发
